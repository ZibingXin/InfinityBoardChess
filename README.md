# 无限棋盘将死

一个基于无限棋盘的将死验证网页，支持拖拽摆子、辅助箭头、预设组合与自定义局面。

## 特性

- 支持平移与缩放功能的无限棋盘
- 拖放棋子及右键拖拽绘制箭头
- 预设棋局组合供将杀技法探索
- 实时将军与将杀状态判定
- 中英文界面切换

## 开始使用

直接用浏览器打开 `index.html`。

# Infinity Board Checkmate (EN)

A web app for exploring checkmate positions on an infinite chessboard.  
Supports drag-and-drop placement, helper arrows, presets, and custom setups.

## Features

- Infinite board with pan & zoom
- Drag-and-drop pieces and right-drag arrows
- Preset combinations for checkmate exploration
- Real-time check / checkmate evaluation
- Chinese / English UI toggle

## Getting Started

Open `index.html` in your browser.
